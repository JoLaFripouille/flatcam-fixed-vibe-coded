# Screenshot evidence

Screenshots in this directory are versioned documentation for visible behavior
changes in this fork.

## Rules

- Keep screenshots in a dated folder: `YYYY-MM-DD-short-topic`.
- Use ordered names: `v0-before...`, `v1-attempt...`, `v2-fixed...`.
- Reference screenshots from `docs/CHANGELOG_VISUAL.md`.
- Do not overwrite old screenshots; add a new version instead.
- Keep private/customer files out of screenshots before publishing.
